Środkowanie zawartości #1

za pomocą margin left i right

HTML

Kod HTML powienien mieć strukturę

<body>
  <div id="wrapper">
    ...tutaj znajduje sie treść którą chcesz wyśrodkować...
  </div>
</body>

CSS

Reguły CSS dla kontenera "wrapper". Wartość "width" według potrzeb, właściwości margin-left i margin-right otrzymują wartość auto.

Kod CSS wygląda tak:

#wrapper { width: 80%; margin-left: auto; margin-right: auto; }
